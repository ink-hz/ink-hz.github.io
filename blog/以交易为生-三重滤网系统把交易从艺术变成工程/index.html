【交易圣经】《以交易为生》三重滤网系统把交易从艺术变成工程

"一个好的交易者看护自己的资金，就像专业的潜水员看护自己的氧气供应一样。" -- 亚历山大·埃尔德（Alexander Elder, M.D.）

亚历山大·埃尔德是一位精神病医生。他在纽约做了多年精神科临床工作之后，转行成为职业交易者。这个看似跳跃的职业转型，恰恰是他最强大的武器——他用医学思维重新定义了交易：先诊断，再开方，最后执行。在他看来，交易不是玄学，不是艺术，而是一套可以拆解、可以量化、可以重复的工程系统。

《以交易为生》自1993年首版以来，被翻译成十余种语言，全球销量超过50万册。这本书的核心贡献不是某一个神奇指标，而是一整套从心理到方法到资金管理的完整交易工程体系。

一、3M体系：交易的三大基石

Elder将交易成功拆解为三个维度，缺一不可：

维度 英文 核心问题 定位
心理 Mind 你能控制自己的情绪吗？ 基础--一切从心理开始
方法 Method 你有可重复的交易系统吗？ 工具--获利手段
资金管理 Money 你能活得足够久吗？ 生存之本--决定是否留在牌桌上

绝大多数散户把全部精力放在"方法"上，四处寻找胜率更高的指标，却忽视了心理和资金管理。Elder的洞察是：方法再好，没有心理纪律就无法执行；资金管理再好，没有方法就只是慢慢亏钱。三者必须协同运作。

五大心法概述

  1. 交易是心理游戏：市场不知道你的存在，你唯一能控制的是自己的行为。大多数亏损不是因为分析方法差，而是因为执行时被情绪绑架。
  2. 与群体保持距离：市场由群体心理驱动——群体在顶部最乐观，在底部最悲观。要想盈利，必须做独立思考的"少数派"。
  3. 先防守后进攻：资金管理不是可选的附加项，而是交易成功的决定性因素。先学会不亏钱，再学会赚钱。
  4. 多重验证，拒绝单一指标依赖：没有任何单一指标能在所有市场环境下可靠工作。用不同类别的指标相互验证，取长补短。
  5. 交易日记是成长的阶梯：坚持记录每笔交易的决策过程、情绪状态和市场环境，才能发现自己的优势和弱点。

二、三重滤网交易系统：核心详解

三重滤网系统（Triple Screen Trading System）是Elder对交易领域最重要的策略贡献，首次发表于1986年的《Futures Magazine》。其设计哲学源自医学的"三重筛查"概念——通过三层递进的筛选，逐层过滤掉不合格的交易机会。

核心原则：每一层筛查如果不通过，就立即放弃，不进入下一层。

时间框架的比喻：潮汐、波浪、涟漪

三重滤网的基础是多时间框架分析，Elder建议时间框架之间保持约5倍的关系。他用海洋的运动来比喻：

  • 潮汐（Tide） = 长期趋势（周线）-- 决定方向
  • 波浪（Wave） = 中期波动（日线）-- 寻找入场时机
  • 涟漪（Ripple） = 短期波动（小时线）-- 确定精确买卖点

在潮汐方向上交易，利用波浪寻找入场点，忽略涟漪中的噪音。

交易者类型 潮汐（长期框架） 波浪（中期框架） 涟漪（短期框架）
波段交易者（默认） 周线图 日线图 4小时图
日内交易者 日线图 1小时图 15分钟图
长线交易者 月线图 周线图 日线图

第一层滤网：市场潮汐（识别趋势方向）

目的：确定长期趋势方向，决定只做多还是只做空。

使用图表：周线图（波段交易者）

指标 参数 用法
MACD（默认推荐） 快线12周、慢线26周、信号线9周 MACD柱状图（Histogram）是核心观察对象
EMA 26周EMA 价格在EMA之上且EMA上升 = 上升趋势

做多条件（周线）：

  1. MACD柱状图从负值区域向上翻转（本周MACD-H > 上周MACD-H），或
  2. MACD线从下向上穿越信号线（金叉），或
  3. 价格位于上升的26周EMA之上

做空条件（周线）：上述条件全部反向。

如果周线趋势向上，只允许寻找做多机会，忽略所有做空信号。如果周线趋势向下，只允许寻找做空机会，忽略所有做多信号。

第二层滤网：市场波浪（寻找回调入场点）

目的：在第一层确认的趋势方向上，等待回调，利用振荡指标识别回调何时结束。

使用图表：日线图（波段交易者）

指标 参数 核心用法
Force Index 2日EMA 捕捉短期回调的极端值
Elder-Ray 13日EMA基准 熊力为负且回升 = 买入信号
随机振荡器 %K=5, %D=3, 平滑=3 超卖区（<20%）回升 = 买入信号

做多时的第二层信号（周线上升，日线回调）：

  1. 等待日线级别的回调下跌
  2. Force Index（2日EMA）跌入负值区域
  3. Elder-Ray熊力指标为负值
  4. 随机振荡器跌至20%以下的超卖区
  5. 当上述指标开始回升时 -- 回调可能即将结束

做空时的第二层信号：上述条件全部反向（日线反弹，指标进入超买区后回落）。

最佳买入机会出现在上升市场短暂下跌的时候；最佳做空机会出现在下跌市场短暂上涨的时候。

第三层滤网：精准入场（确定具体买卖点）

目的：在前两层确认方向和时机后，在短期时间框架上确定精确的入场价格。

入场方法：追踪止损单（Trailing Stop Order）

做多入场：

  1. 在前一日最高价之上一个tick设置追踪买入止损单（Buy Stop）
  2. 如果价格上涨触发止损单，自动入场做多
  3. 如果价格继续下跌未触发，将止损单下移至新一日最高价之上一个tick
  4. 持续追踪，直到触发入场或周线趋势改变

做空入场：在前一日最低价之下一个tick设置追踪卖出止损单（Sell Stop），逻辑反向。

滤网层级 时间框架 核心指标 关键参数 决策内容

完整A股交易流程示例

场景：波段交易者，交易某A股股票，账户10万元

第一步：周线图分析（第一层滤网 -- 潮汐）

  • 打开周线图，查看MACD指标（12, 26, 9）
  • 发现MACD柱状图上周为-0.15，本周为-0.08，正在从负值区域向上收窄
  • MACD线在信号线下方但差距缩小，即将形成金叉

结论：周线趋势即将转为上升，准备寻找做多机会

第二步：日线图分析（第二层滤网 -- 波浪）

  • 切换到日线图，查看随机振荡器（5, 3, 3）
  • 股票在过去5天持续回调，随机振荡器跌至15%（超卖区）
  • Force Index（2日EMA）为负值，且开始回升
  • Elder-Ray熊力指标为-0.8，但正在从-1.2回升

结论：日线回调即将结束，准备入场做多

第三步：短期图表执行（第三层滤网 -- 涟漪）

  • 切换到60分钟图
  • 前一日最高价为25.30元
  • 设置追踪买入止损单：25.31元（25.30 + 0.01）
  • 设定止损位：日线近期低点24.50元下方，取24.40元

若价格涨至25.31元，自动入场做多

第四步：风控与仓位计算

账户总资金：   100,000 元
2%风险上限：   100,000 x 2% = 2,000 元
入场价：       25.31 元
止损价：       24.40 元
每股风险：     25.31 - 24.40 = 0.91 元
可买入股数：   2,000 / 0.91 = 2,197 股
取整（按A股100股整手）：2,000 股（20手）
所需资金：     2,000 x 25.31 = 50,620 元
触发止损亏损： 2,000 x 0.91 = 1,820 元（账户的1.82%，未超2%）

第五步：出场

  • 移动止损：随着价格上涨，逐步将止损上移至近期日线低点
  • 出场信号：日线MACD柱状图开始下降，或Force Index出现顶背离

三、资金管理铁律：2%规则和6%规则

Elder反复强调：资金管理不是可选的附加项，而是交易成功的决定性因素。他提出了两条铁律。

铁律一：2%规则（单笔风险上限）

定义：每一笔交易的最大允许亏损不超过账户总资金的2%。

注意：这2%不是指用2%的资金去买股票，而是指如果止损被触发，你最多只亏2%。

最大允许亏损金额 = 账户总资金 x 2%
可买入股数 = 最大允许亏损金额 / (入场价 - 止损价)

计算示例：

账户总资金：   500,000 元
2%风险上限：   500,000 x 2% = 10,000 元
入场价：       25.50 元
止损价：       24.50 元
每股风险：     25.50 - 24.50 = 1.00 元
可买入股数：   10,000 / 1.00 = 10,000 股
所需资金：     10,000 x 25.50 = 255,000 元（占账户51%）
触发止损亏损： 10,000 元（账户的2%，绝不超过）

止损位由市场结构决定（支撑位/阻力位），而非由2%规则决定。2%规则决定的是仓位大小，不是止损位。

铁律二：6%规则（月度风险上限）

定义：在任何一个月份内，所有持仓的总风险敞口加上已实现的月度亏损，不超过账户总资金的6%。

具体操作：

  1. 每月初重置计数器
  2. 每开一笔新仓，将风险金额加入月度风险总计
  3. 当月度风险总计达到账户的6%时，停止开新仓
  4. 如果当月实际亏损达到6%，当月停止所有交易，下个月重新开始

计算示例：

账户总资金：     500,000 元
6%月度上限：     500,000 x 6% = 30,000 元

第一笔交易风险： 10,000 元（2%） → 月度风险余额：20,000 元
第二笔交易风险：  8,000 元       → 月度风险余额：12,000 元
第三笔交易风险： 12,000 元       → 月度风险余额：0 元，停止开新仓

如果前两笔都止损（亏损18,000元）：
  月度实际亏损 = 18,000 元
  第三笔风险 12,000 元 → 18,000 + 12,000 = 30,000 = 6%
  可以开仓，但不再增加任何新仓位

2%规则管单笔，6%规则管全局。两条规则配合使用，确保即使在最糟糕的月份，账户也不会遭受毁灭性打击。

四、其他策略简述

冲动系统（Impulse System）

由13日EMA（趋势方向）和MACD柱状图（动量方向）组合而成。两者同向上 = 绿色冲动（看涨）；两者同向下 = 红色冲动（看跌）；方向不一致 = 蓝色/中性（不交易，收紧止损）。这个系统可直接编码为指标，适合作为交易系统的过滤器。

Elder-Ray系统（牛力/熊力）

Elder独创的震荡指标。牛力 = 当日最高价 - 13日EMA；熊力 = 当日最低价 - 13日EMA。做多入场条件：EMA上升 + 熊力为负且回升（空头力量减弱）。做空入场条件反向。出场信号为牛力/熊力出现背离。

强力指数（Force Index）

量价结合振荡指标。Force Index = (今日收盘价 - 昨日收盘价) x 今日成交量。2日EMA用于捕捉短期回调结束信号，13日EMA用于检测中期背离。价格创新高但13日Force Index未创新高 = 看跌背离；反向同理。

五、交易心理学：7种心理陷阱及应对

Elder的精神病学背景使他深刻揭示了一个事实：交易的本质不是与市场搏斗，而是与自己的内心搏斗。他总结了7种最常见的心理陷阱：

心理陷阱 典型表现 Elder的应对建议
过度自信 连续盈利后加大仓位、放松止损 连赢后更要严格遵守规则，不改变系统
赌徒心理 亏损后急于翻本、频繁交易 启动6%规则强制休息，远离市场
后悔与幻想 "要是当时没卖就好了" "肯定会涨回来的" 专注当下决策，不浪费精力在后悔上
权威依赖 盲目跟随"大师"推荐 建立自己的交易系统，独立做决策
证实偏差 只看到支持自己观点的信息 主动寻找反面证据，定期审视持仓理由
害怕错过（FOMO） 看到别人赚钱就急于入场 "错过机会不亏钱，错误入场才亏钱"
希望心理 亏损时不认赔，期待"会回来的" 止损是保护，不是认输

市场并不知道你的存在。你无法影响它。你唯一能控制的是自己的行为。

写在最后

《以交易为生》的真正价值，不在于教会你一个指标或一个公式，而在于帮你建立一套工程化的交易思维。

三重滤网系统的精髓可以用三个词概括：方向、时机、执行。潮汐告诉你往哪个方向走，波浪告诉你什么时候下水，涟漪帮你精确定位入水点。每一层都过滤掉一批不合格的机会，最终留下的就是高质量的信号。

而2%和6%这两条铁律，确保你即使在连续判断失误的情况下，也不会被市场淘汰。正如Elder所说：

先学会不亏钱，再学会赚钱。资金管理不是锦上添花，而是生死线。

交易的对手从来不是市场，而是你自己。当你能像工程师一样系统化地执行交易计划，像精神病医生一样冷静地观察自己的情绪，你就有机会真正做到——以交易为生。
