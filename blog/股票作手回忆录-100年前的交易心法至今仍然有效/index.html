【交易圣经】《股票作手回忆录》100 年前的交易心法至今仍然有效

2026/5/28
#股票交易 #交易心法

引言：百年华尔街的一个幽灵

1929年10月29日，黑色星期二。纽约证券交易所里一片哀嚎，无数人倾家荡产，从高楼一跃而下。然而在这一天，有一个人通过做空赚取了1亿美元——相当于今天的15亿美元。

他叫杰西·利弗莫尔（Jesse Livermore，1877-1940）。

14岁在波士顿的投机行里用5美元开始第一笔交易，30岁已在华尔街呼风唤雨。他一生经历了四次破产、四次东山再起。1907年银行危机中做空赚得300万美元，1929年大萧条中做空获利1亿美元。他的交易手法被巴菲特、索罗斯、芒格公开推崇，他的话被现代量化交易的先驱们反复引用。

1923年，财经作家埃德温·勒菲弗以利弗莫尔为原型，写下了《股票作手回忆录》。一百年过去了，华尔街换了无数面孔，交易工具从纸带变成了算法，但这本书仍然是每一位交易者的必读经典。

"华尔街永远不会改变，因为人性永远不会改变。"（Wall Street never changes, because human nature never changes.）

为什么？因为无论技术如何进化，驱动市场涨跌的根本力量始终是人的贪婪与恐惧。利弗莫尔用一生的代价换来的交易智慧，本质上是对人性的洞察——而人性，从未改变。

一、五大核心心法

心法一：顺势而为——沿着"最小阻力线"交易

"价格和其他一切事物一样，沿着最小阻力线运动。它们会做最容易做的事情。"（Prices, like everything else, move along the line of least resistance. They will do whatever comes easiest.）

利弗莫尔的核心交易哲学极其简洁：不要预测市场方向，而是识别市场的"最小阻力线"——即供需力量的整体对比方向。当多头力量大于空头力量时，价格上涨是最容易的事情；反之亦然。

他年轻时在波士顿的投机行里，靠"盘感"赚到了第一桶金。但当他带着全部身家来到纽约的正规交易所时，却很快亏光了。原因是他习惯了频繁交易、试图抓住每一个小波动。后来他领悟到：真正赚大钱不是靠频繁进出，而是靠识别大趋势并坚持持有。

实战要点：

  • 不预测，只跟随：观察关键价位被突破后的价格行为，让市场告诉你它的方向
  • 多头市场只做多，空头市场只做空：不要在明确的下跌趋势中抄底
  • 每日收盘评估：判断"最小阻力线"的方向是否发生了改变

心法二：关键点突破——在趋势的起点进场

"从来不是我的思考让我赚了大钱，而是我的坐等。明白吗？我的坐等不动！"（It was never my thinking that made the big money for me. It always was my sitting. Got that? My sitting tight!）

利弗莫尔的"关键点"（Pivotal Point）系统是他识别重大趋势转折的核心工具。关键点是价格在长期盘整后突破的价位，代表供需力量发生了根本性转变。

关键点分为两种：

  1. 主要关键点：标志着新一轮大趋势的开始，出现在长期盘整或下跌之后的突破
  2. 次级关键点：趋势运行过程中的回调低点或反弹高点，用于加仓

利弗莫尔的操作方式是：在关键点附近建立初始头寸，然后观察价格走势。如果突破后继续朝预期方向运动，说明判断正确，继续加仓；如果突破后迅速回落，立即离场。赚钱的头寸从一开始就是盈利的——这是他最核心的判断标准。

心法三：金字塔加仓——赢者加注，输者离场

"亏损在我承受之后从不困扰我，我一夜之间就忘记了。但犯错——不承受亏损——才是毁灭所有人的原因。"（A loss never bothers me after I take it. I forget it overnight. But being wrong — not taking the loss — that is what destroys people.）

这是利弗莫尔赚到大钱的关键秘诀。他从不一次性满仓，而是分批建仓，每次只在盈利的头寸上加码，且加仓量递减。

为什么递减？因为越往后面，你的平均成本越高，风险也越大。递减加仓确保平均成本始终处于有利位置，即使价格出现一定回调，你仍然盈利。

最核心的一条铁律：亏损的头寸永不加仓。不摊平成本，不抱幻想。利弗莫尔曾多次因为摊平成本而爆仓，每一次都用真金白银验证了这条规则的代价。

心法四：耐心等待——一年中真正的好机会只有几次

"在我所有的投资生涯中，我发现大钱从来不是在买入或卖出时赚到的。大钱是在等待中赚到的。"（Throughout all my years of investment I've found that the big money was never made in the buying or the selling. The big money was made in the waiting.）

利弗莫尔在书中反复强调：市场中大部分时间是不适合交易的。真正赚大钱的机会一年只出现几次。交易者最大的敌人往往是自己——过度交易、不耐烦、害怕错过（FOMO）。

他本人曾多次因为不耐心而亏损巨款。1906年他在空头市场中过早做多棉花期货，亏掉了之前的全部利润。这次教训让他刻骨铭心：持有现金本身就是一种主动决策，空仓不是"什么都没做"，而是在等待最佳的出击时机。

心法五：独立思考——拒绝从众

"只有当我自己独立思考时，我才是正确的。"（I was only right when I was doing my own thinking.）

利弗莫尔的整个投机生涯都在强调独立思考。他认为，市场中大多数人是亏钱的，如果你跟着大多数人走，结果必然也是亏损。他反复警告不要听信"内幕消息"——因为当你收到消息时，信息已经反映在价格中了。

在1907年恐慌前夕，利弗莫尔通过独立分析判断出市场即将崩盘，果断做空。而当时华尔街的大多数人还在听信"股市永远上涨"的乐观论调。结果他一战成名，赚得300万美元。

二、六大可量化策略详解

策略一：关键点突破策略

这是利弗莫尔最核心的趋势跟随策略，适用于股票、期货、外汇等流动性充足的市场。

参数规则
盘整期识别价格在至少3周以上的区间内横盘整理，区间幅度不超过15%
关键点定义区间上沿为"关键买入点"，区间下沿为"关键卖出点"
突破确认收盘价突破关键点，且突破幅度 >= 区间宽度的3%
成交量确认突破日成交量 >= 过去20日平均成交量的150%
首次建仓突破确认后，使用计划总仓位的25%
止损位设在突破关键点下方2-3%
目标位不设固定目标，使用移动止损跟随趋势
 
// 关键点突破策略伪代码
INPUT: price_series, volume_series, consolidation_min_weeks=3, range_max_pct=15

// Step 1: 识别盘整区间
consolidation_range = find_consolidation(price_series, min_weeks=3, max_range_pct=15)
IF consolidation_range NOT FOUND:
    RETURN "等待盘整形成"

upper_pivot = consolidation_range.high
lower_pivot = consolidation_range.low

// Step 2: 等待突破
IF close > upper_pivot * 1.03:  // 突破上沿3%
    avg_vol_20 = SMA(volume, 20)
    IF volume > avg_vol_20 * 1.5:  // 成交量放大50%
        position_size = total_capital * 0.25 * risk_per_trade / (close * 0.03)
        ENTER_LONG(position_size)
        stop_loss = upper_pivot * 0.97  // 止损在关键点下方3%

策略二：金字塔加仓策略

在趋势确认后，逐步加大仓位。每次加仓量递减，形成金字塔结构。

步骤价格条件加仓比例止损调整
首次建仓关键点突破确认总计划的25%关键点下方2-3%
第一次加仓价格上涨1-2%总计划的20%移至首次建仓价位
第二次加仓价格再上涨1-2%总计划的15%移至第一次加仓价位
第三次加仓价格再上涨1-2%总计划的10%移至第二次加仓价位
最终仓位—累计70%—

核心约束：每次加仓前，前一笔必须处于盈利状态；加仓量必须递减；如果价格回撤到总平均成本，立即清仓全部头寸；绝不在亏损头寸上加仓。

// 金字塔加仓策略伪代码
tiers = [
    {size_pct: 0.25, price_move: 0.00},   // 首次
    {size_pct: 0.20, price_move: 0.01},   // +1%
    {size_pct: 0.15, price_move: 0.02},   // +2%
    {size_pct: 0.10, price_move: 0.03},   // +3%
]

IF current_tier < len(tiers) - 1:
    next_tier = tiers[current_tier + 1]
    price_gain = (current_price - entry_price) / entry_price

    IF price_gain >= next_tier.price_move:
        new_size = total_position_size * next_tier.size_pct
        ADD_POSITION(new_size)
        UPDATE_STOP_LOSS(breakeven + small_buffer)

    IF current_price <= calculate_avg_cost():
        CLOSE_ALL_POSITIONS()  // 回到平均成本，全部清仓

策略三：量价分析策略

利弗莫尔时代的"纸带阅读"（Tape Reading）是今天的量价分析。他用这种方法判断市场处于四个阶段中的哪一个：吸筹、拉升、派发、下跌。

阶段价格特征成交量特征关键信号
吸筹长期下跌后横盘，波动收窄萎缩至地量，偶尔放量但不跌利空消息不再导致下跌
拉升突破区间，创新高和新低上涨放量，回调缩量突破关键点时显著放量
派发涨势放缓，宽幅震荡上涨缩量，下跌放量利好消息不再推高价格
下跌跌破区间，创新低和新低下跌放量，反弹缩量反弹无法突破前高

策略四：10%止损规则

利弗莫尔的止损哲学极其严格：任何单笔交易亏损不超过总资金的2%，任何持仓亏损达到10%无条件清仓。

// 止损规则伪代码
hard_stop = entry_price * 0.90           // 亏损10%强制止损
max_risk_amount = total_capital * 0.02   // 2%资金风险
price_stop = entry_price - (max_risk_amount / position_size)
actual_stop = MAX(hard_stop, price_stop)

// 利润保护规则
IF unrealized_profit_pct >= 0.05:
    trailing_stop = entry_price * 1.01   // 移至盈亏平衡点上方1%

IF unrealized_profit_pct >= 0.10:
    trailing_stop = MAX(unrealized_profit_pct * 0.50, entry_price * 1.03)

策略五：50%现金储备原则

参数规则
最大持仓比例总资金的50%（至少保留50%现金）
单只股票上限总资金的10-15%
同一板块上限总资金的25%
集中度要求最佳机会可集中至50%，但不超限

利弗莫尔亲身验证了这条规则的必要性：他多次因为满仓而在市场崩盘时损失惨重。永远有弹药在关键时刻出击，比满仓被套好一万倍。

策略六：自上而下分析框架

利弗莫尔坚持从宏观到微观的三层分析：第一层判断大盘是牛市还是熊市；第二层确认强势板块；第三层在强势板块中寻找关键点突破的个股。他只做强势板块中的强势股，从不碰弱势板块。

分析层级优先级核心规则
市场整体趋势最高牛市做多，熊市做空，震荡观望
行业板块方向其次只做强势板块，不做弱势板块
个股价格行为第三等待关键点突破后进场

四、交易心理学四大洞察

洞察一：恐惧与贪婪的永恒循环

"投机者的致命敌人是：无知、贪婪、恐惧和希望。"（The speculator's deadly enemies are: ignorance, greed, fear and hope.）

利弗莫尔深刻理解，市场中每一次繁荣与崩溃背后，都是同样的心理力量在驱动。价格持续上涨时，贪婪驱使人们纷纷入场，风险最大；价格开始暴跌时，恐惧引发恐慌性抛售，机会最大。

更反直觉的是：在亏损时人们抱有"希望"（价格会回来的），在盈利时人们感到"恐惧"（利润会消失）。正确的做法恰恰相反——亏损时要果断行动，盈利时要耐心持有。

洞察二：从赌徒到大师的五级跃迁

利弗莫尔的心理成长经历了五个阶段：

  1. 赌徒阶段：年轻时靠直觉和盘感赚钱，自信满满
  2. 学费阶段：到正规交易所后，发现方法不够用，多次爆仓
  3. 学习阶段：开始理解市场心理和供需关系，从盘感转向系统分析
  4. 成熟阶段：学会耐心、纪律和风险管理，在关键点交易
  5. 大师阶段：1929年大崩盘中做空获利1亿美元，达到巅峰

他最大的心理突破是学会了"坐等"（sitting tight）。赚大钱不在于频繁交易的技巧，而在于在正确时机建仓后，耐着性子持有不动，直到趋势完成。

洞察三：自我认知的六大心理陷阱

"股市只有一面，不是牛市的一面或熊市的一面，而是正确的一面。"（There is only one side to the stock market; and it is not the bull side or the bear side, but the right side.）

利弗莫尔告诫交易者必须克服六大心理障碍：

  • 过度自信：连续盈利后放松纪律，以为自己不会犯错
  • 报复性交易：亏损后急于翻本，加大仓位，导致更大亏损
  • 害怕错过：看到别人赚钱而焦虑入场，往往买在顶部
  • 确认偏误：只看到支持自己观点的信息，忽略反面信号
  • 沉没成本：因为已经亏损而不愿止损，越亏越多
  • 外部依赖：依赖他人的意见和"内幕消息"，丧失独立判断

洞察四：市场是一面镜子

利弗莫尔深刻认识到，市场会放大你性格中的每一个弱点。如果你急躁、贪婪、恐惧或固执，市场都会精准地惩罚你。终极法则是：了解自己比了解市场更重要。在市场中赚钱，你需要做到大多数人做不到的事情——控制自己的情绪，按照纪律行动。

五、A股市场适用性分析

利弗莫尔的原则诞生于100年前的美股市场，今天在A股中是否适用？答案是：核心原则高度适用，但具体参数需要调整。

高度适用的原则

  • 关键点突破：A股涨跌停板制度为突破提供了天然参考，可结合龙虎榜数据
  • 量价分析：A股主力资金进出通过成交量暴露，使用换手率替代绝对成交量
  • 自上而下分析：A股受政策影响极大，"政策底"和"市场底"可纳入趋势分析
  • 50%现金储备：A股波动性大、散户比例高，建议提高至60-70%

需要调整的参数

利弗莫尔原始参数A股适配参数调整原因
突破3%确认突破3%或触及涨停板涨跌停限制使价格运动模式不同
盘整3周以上盘整2-4周 + 布林带收窄A股热点轮动快，周期适当缩短
成交量放大50%换手率 > 5% + 量比 > 2个股流通盘差异大
10%止损7-8%止损（预防一字跌停）A股存在跌停无法卖出的风险
金字塔加仓1-2%间距金字塔加仓2-3%间距A股波动更大，间距需加宽
50%现金储备60-70%现金储备A股极端波动（千股跌停）更频繁

A股特色调整

  • 做空限制：A股融券做空困难，熊市中可使用股指期货（IF、IC、IH）或50ETF看跌期权，或直接空仓等待
  • T+1制度：当日买入次日才能卖出，试仓规模控制在更小比例（10%以内）
  • 涨跌停板：止损策略可能遇到"一字跌停"无法卖出的情况，应在异常放量滞涨时提前减仓
  • 政策因子：在趋势判断中加入政策面分析（货币政策、行业政策、监管态度）

利弗莫尔的一生是一个极具戏剧性的故事：他四次破产、四次东山再起，最终在1940年以自杀结束了自己的生命。他留给世界的，不是一套可以无脑复制的交易公式，而是对市场本质和人性本质的深刻洞察。

正如利弗莫尔所说：

The game taught me the game. And it didn't spare the rod while teaching.
